# Red Panda UI

## Getting Started

1) Install Red Panda UI into your React project.
```bash
npm i ui-redpanda
```
2) Import your desired component and the styles.

```typescript
import { Button } from 'ui-redpanda';
import "ui-redpanda/dist/style.css";
```
3) Use the component.

```tsx
import { Button } from 'ui-redpanda';
import "ui-redpanda/dist/style.css";

const export App = () => {
    return <Button>Click Me</Button>
}
```
